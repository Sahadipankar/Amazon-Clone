# Amazon Clone

![Amazon Clone Screenshot](https://raw.githubusercontent.com/Sahadipankar/Amazon-Clone/main/Demo.png)

## About This Project

This is my first web development project, built in early 2024. It's a front-end clone of Amazon's homepage, created to practice HTML and CSS fundamentals.

## Technologies Used

- HTML5
- CSS3
- Font Awesome Icons

## Features

- **Responsive Navigation Bar** - Includes logo, delivery location, search functionality, language selector, and cart
- **Product Categories** - Multiple product category boxes showcasing different departments
- **Hero Section** - Banner with location-based messaging
- **Footer** - Multi-column footer with links organized by category

## Project Structure

```
├── index.html          # Main HTML file
├── style.css           # Styling
└── README.md          # Project documentation
```

## What I Learned

- HTML semantic structure
- CSS Flexbox layouts
- Styling navigation bars and footers
- Organizing product card layouts
- Working with Font Awesome icons
